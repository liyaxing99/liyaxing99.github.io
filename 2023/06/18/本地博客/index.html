如何搭建一个本地博客

软件安装

node 博客配置工具

1.安装网站：nodejs.org

2.安装完成后右键 win图标 打开 “Windows PowerShell（管理员）”

3.输入

查看安装情况及版本

4.安装npm国内镜像版本

输入 npm install -g cnpm –registry=https://registry.npm.taobao.org

5.安装完成输入 cnpm -v 查看安装情况及版本号

安装hexo博客

1.输入 cnpm install -g hexo-cli

2.输入 hexo -v 查看安装情况及版本号

搭建博客

1.建立一个存放博客文件的文件夹

输入mkdir blog创建文件夹，也可手动新建文件夹

2.进入blog文件夹

cd blog/

3.生成博客

hexo init

启动博客

hexo s
